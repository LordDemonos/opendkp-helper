OpenDKP Helper Settings

🎚️ Mode

Choose how the extension behaves. Raid Leader mode enables raid leader tools and stronger alerts. Raider mode focuses on personal alerts. Auto-Bid is available in both modes.

🤖 Auto-Bid

Automatic API bidding on active auctions. Available in both Raid Leader and Raider modes — keep an opendkp.com tab open while enabled.

Sign in with your OpenDKP account (same as the website). Credentials are stored locally and shared with Raid Leader API tools if you switch modes.

Uses guild subdomain and API sign-in from OpenDKP API (raids) below.

While an opendkp.com tab is open, polls active auctions and bids in increments until you are in a winning slot (top bid for single items, top N for quantity N), your max DKP for that item is reached, or you run out of DKP. Character rank is loaded from OpenDKP when you refresh characters. Rank bid limits (e.g. Raid Alt max 400) are read from the site Bid Rules panel and enforced in settings and auto-bid. When your character is called as the winner for a matching item, that rule is automatically disabled (re-enable the checkbox to bid on the same item again).

🎨 Appearance

Choose light or dark theme for the extension (settings and popup).

    ⚔️ Loot Parser
    
            
    
                Firefox restricts extensions from keeping live access to local files. To reliably parse loot:
                open the Loot Monitor window, choose your EQ log once, and leave the window open.
                It checks every few seconds and pushes new loot lines into the popup automatically.
            
    
             
    
                In game, type your identifier before pressing 'Link Loot' (Zeal Feature with compatible UI). The parser will detect log lines containing your identifier and push the items to the popup.
            
    
            
    
            
    
                
            
    
            
    Close the monitor window to stop; reopen it to resume. Nothing is uploaded—all processing stays local.

    📦 Backup & Restore
    
            
    
                Export all settings to a file — reminders, RaidTick, OpenDKP API (guild, username, roster/pools cache, tick definitions, auction defaults), item watchlist, EQ log rules, and custom sounds. Watchlist and API fields auto-save as you type. Download a backup before reloading or updating the extension. Restore after reinstalling or when testing new versions.
            
    
            
    
                
                
            
    
            
    
                
                
    
                    Password and JWT tokens are stored in plain text in the backup file. Store the file somewhere safe—anyone with the file can sign in as you.
